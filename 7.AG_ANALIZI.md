\# 🌐 7. Ağ Analizi ve Gizli Trafik Analizi (Forensics)



\*\*Görev:\*\* Uygulamanın arka plandaki gizli trafiğini ve API iletişimini incelemek.



\### 7.1. DevTools ve Trafik Analizi

Gitleaks, uzak repoları tararken GitHub API'si ile iletişime geçer. Bu süreçte oluşan trafiği \*\*Browser DevTools\*\* ve \*\*Burp Suite\*\* üzerinden analiz ettim.

\* \*\*HAR Dosyası Analizi:\*\* Tarama sırasında oluşan HTTP isteklerini (Request/Response) `.har` formatında dışa aktararak, API anahtarlarının veya session bilgilerinin "Network Layer" (Ağ Katmanı) üzerinde sızıp sızmadığını denetledim.



\### 7.2. Gizli Trafik ve Adli Bilişim

Uygulamanın arka planda gitmediği gizli sunucular (C2 - Command \& Control) olup olmadığını doğrulamak için ağ paketlerini inceledim. Bu, derste işlediğimiz "Uygulama arka planda ne yapıyor?" sorusuna bir adli bilişim (forensics) yanıtıdır.
